Dropbox

Dropbox is one of the most popular file hosting services which offers cloud storage, client software and synchronisation of files. The application allows the users to create specific folders on their computers.These folders are accessible through mobile phones and websites. The dropbox synchronises itself in such a way that it appears as the same folder with the same contents on any system, be it a laptop, desktop or phone.
